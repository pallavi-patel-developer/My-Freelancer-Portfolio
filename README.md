# Pallavi Patel - Full Stack AI Freelancer Portfolio

A high-end, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features
- Dynamic Hero Section with floating tech icons.
- Advanced Custom Cursor with trailing dot effect.
- Serverless Contact Form integrated with Web3Forms.
- Fully Responsive Design for mobile and desktop.
- Interactive Service and Project cards.

## Deployment
This project is optimized for deployment on **Render** as a Static Site.
